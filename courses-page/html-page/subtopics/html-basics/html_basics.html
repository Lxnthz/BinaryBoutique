HTML Basics

In this chapter we will show some basic HTML examples.

Don't worry if we use tags you have not learned about yet.

HTML Documents.

All HTML documents must start with a document type declaration: <!DOCTYPE html>

The HTML document itself begins with <html> and ends with </html>

The visible part of the HTML document is between <body> and </body>.

Example: 

                    
                        <!DOCTYPE html>
                        <html>
                            <body>
                            
                                <h1>My First Heading</h1>
                                <p>My first paragraph.</p>
                            
                            </body>
                        </html>

Basic Structure of an HTML Web Page

The fundamental layout of an HTML page is outlined beneath, featuring key elements such as the doctype declaration, HTML, head, title, and body elements. These components collectively constitute the foundational structure upon which all web pages are constructed.

HTML Basic Tags

  • <!DOCTYPE html> – A doctype, or document type declaration, is an instruction informing the web browser about the markup language used in the current page. It's not a tag or element and isn't case-sensitive.
  • <html> – This tag establishes the root element of an HTML document, indicating to the browser that the content is in HTML. It serves as the second outer container, encompassing all other elements.
  • <head> – Used to define the head section of an HTML document, this tag contains information related to the document. Elements within the head tag are not visible on the webpage's front end.
  • <body> – The body tag encapsulates all the visible content of a webpage. Essentially, the body content is what the browser displays on the frontend.

Example: 

                    
                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Copyable Code Snippet</title>
                    </head>
                    <body>
                        <p>Hello, World!</p>
                    </body>
                </html>
